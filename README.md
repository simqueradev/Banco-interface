# "Banco" fictício em C#
Um projeto que faz transações monetárias fictícias, usando C# e .NET, sem banco de dados (alocação de dados na memória).

## Instalação
*Pré-requisitos:*

-.NET SDK 8.0 ou superior  
-Visual Studio Code com a extensão C#  

1. Verifique a instalação do .NET:  
dotnet --version  

2. Clone o repositório:  
git clone https://github.com/fellipesiqueiradev/Banco-interface.git  
cd Banco-interface  
cd teste16_interfaceteste  

3. Execute o "Program.cs":  
dotnet run
